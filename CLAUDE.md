# Design Context

This is a **brand**-register project (artist portfolio — the interface IS the
work's first impression). See **PRODUCT.md** for the full strategic brief and
**DESIGN.md** for the visual token system. Flow commands read both automatically.

Five strategic principles guide all design work:

1. **The work speaks first** — imagery leads every screen; words support.
2. **Catalog gravitas, gallery immersion** — editorial art-book AND cinematic
   dark room, held in tension.
3. **Curate, never dump** — 155 works given hierarchy and breathing room.
4. **Restraint earns the drama** — quiet chrome so the art can be loud.
5. **Bilingual without compromise** — Italian authoritative, English first-class.

Avoid: SaaS/template looks, sterile white-cube minimalism, Instagram-feed
flatness, effect-for-its-own-sake spectacle. Accessibility: WCAG AA where it
counts, atmosphere first; full `prefers-reduced-motion` support.
